Logical Model: Case Report Logical Model

Defining URL:https://www.openhie.org/fhir/hiv/StructureDefinition/model-hiv-case-report
Version:0.2.0
Name:HIVCaseReport
Status:Draft as of 2020-11-12T13:10:28+00:00
Definition:

This is the logical model for the HIV case report. This report contains the relevant data, which can be acquired from manually filled questionnaires, importing spreadsheet content, or extracting data from an EHR system.

Publisher:openHIE
Source Resource:XML / JSON / Turtle
